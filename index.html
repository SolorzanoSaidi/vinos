Claro, aquí tienes un ejemplo básico de código HTML para una página web estática sobre los tipos de vinos: ```html Tipos de Vinos

Tipos de Vinos

  • Vino tinto:
    El vino tinto se produce a partir de uvas tintas y se caracteriza por su color rojo oscuro. Suele tener un sabor más intenso que otros tipos de vino y marida bien con carnes rojas y quesos curados.
  • Vino blanco:
    El vino blanco se elabora principalmente con uvas blancas o tintas de pulpa blanca. Tiene un color amarillo claro y su sabor puede variar desde seco hasta dulce. Es una buena opción para maridar con pescados, mariscos y platos ligeros.
  • Vino rosado:
    El vino rosado se obtiene de uvas tintas, pero el contacto con las pieles es menor que en el caso del vino tinto. Tiene un color rosado y suele tener un sabor fresco y afrutado. Es una buena elección para acompañar ensaladas, platos de pasta y aves.
``` Este código crea una página web con tres tipos de vinos (tinto, blanco y rosado) y una breve descripción de cada uno. Puedes personalizar y ampliar este código según tus necesidades y agregar más estilos o información sobre otros tipos de vinos.
